  1. Acesse supabase.com e crie uma conta gratuita
    2. 
                    
  2. Crie um New Project (escolha uma senha e regiao)

    4. 
                    
  4. Va em SQL Editor (menu lateral) e cole o SQL abaixo, depois clique Run
    5. 
                    
  5. Va em Project Settings > API (menu lateral)
    6. 
                    
  6. Copie o Project URL e a anon public key
    7. 
                    
  7. Cole nos campos abaixo e salve
    8. 
                    
  8. Importante: Em Authentication > Providers > Email, desmarque "Confirm email" para login imediato
    9. 
                

                

                    
SQL para criar a tabela (copie e cole no SQL Editor):

                    
CREATE TABLE hubias_sync (
  id BIGSERIAL PRIMARY KEY,
  user_id UUID NOT NULL DEFAULT auth.uid() REFERENCES auth.users(id) ON DELETE CASCADE,
  key TEXT NOT NULL,
  value TEXT,
  updated_at TIMESTAMPTZ DEFAULT NOW(),
  UNIQUE(user_id, key)
);

ALTER TABLE hubias_sync ENABLE ROW LEVEL SECURITY;

CREATE POLICY "Users manage own data" ON hubias_sync
  FOR ALL USING (auth.uid() = user_id)
  WITH CHECK (auth.uid() = user_id);
